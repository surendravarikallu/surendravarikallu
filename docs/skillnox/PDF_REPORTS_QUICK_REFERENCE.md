## 🚀 **How to Generate Reports**

### **Step 1: Access Reports**
1. Go to **Admin Dashboard**
2. Click **"Export Reports"**
3. Select **Contest** from dropdown
4. Choose **Report Type**

### **Step 2: Generate PDF**
1. Click **"Generate Report"**
2. Wait for processing (usually 2-5 seconds)
3. PDF downloads automatically
4. File saved with professional naming

### **Step 3: Use Reports**
- **Print** for physical distribution
- **Email** to stakeholders
- **Archive** for record keeping
- **Share** with students/parents
